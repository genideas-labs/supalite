# 제약과 트레이드오프

SupaLite는 가볍고 직접적인 PostgreSQL 클라이언트를 목표로 합니다.
아래 항목은 현재의 제약 또는 비목표입니다.

- Supabase 완전 대체는 아닙니다: Auth/Storage/Realtime는 범위 밖입니다.
- CLI/타입 출력은 Supabase CLI와 1:1 호환이 아닙니다.
- 함수 정의 덤프는 민감 SQL이 포함될 수 있어 공개 저장소에 주의가 필요합니다.
- 스키마 키 케이스 변환을 사용할 경우 DB 실제 식별자와 불일치 위험이 있습니다.
- ORM 기능(관계 모델링/중첩 쓰기/마이그레이션)은 외부 도구로 해결합니다.
